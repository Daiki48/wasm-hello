# WASM HELLO

Minimum configuration using wasm-bindgen.

ref - https://rustwasm.github.io/wasm-bindgen/examples/hello-world.html

# Setup

```shell
$ git clone https://github.com/Daiki48/wasm-hello.git
$ pnpm install
$ pnpm install -D wasm-pack
```

# Run

```shell
$ pnpm run serve
```

`http://localhost:8080`
